# The Golden Whistle


What is this? A website about the World Cup that will show what the world cup is and information on what countries qualified this year.
Why does it exist? Just to have some information on the World Cup for anyone who is interested in learning more about it.
What tools did I use? I used GitHub, Gemini, and netlify.app so far.
How to visit it? https://thegoldenwhistle.netlify.app/
