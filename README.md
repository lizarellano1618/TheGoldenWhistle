# The Golden Whistle


This is a website about the World Cup that will show what the world cup is and information on what countries qualified this year.
This exists just to have some information on the World Cup for anyone who is interested in learning more about it. So far I have used GitHub, Gemini, and netlify.app in order to get this site together.
The link to the the site is https://thegoldenwhistle.netlify.app/.
